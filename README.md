# NeuroObs by wail

Application d'Observation Neurologique Interactive.

## Technologies
- React
- Vite
- Tailwind (via CSS variables)

## Installation
```bash
npm install
npm run dev
```

## Déploiement (Vercel)
Ce projet est configuré pour être déployé directement sur Vercel.
Importez simplement le dépôt GitHub.
